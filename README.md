# 프로젝트 관리 가이드

이 저장소의 각 폴더는 독립된 프로젝트로 관리됩니다.

## 독립된 경로 사용 안내

각 프로젝트(예: `naverapieda`, `seoul_pops`)를 작업할 때는 다음 사항을 준수해 주세요:

1. **독립된 경로 설정**: 각 프로젝트 폴더를 작업 디렉토리(Working Directory)로 설정하여 상대 경로가 꼬이지 않도록 합니다.
2. **가상환경 관리**: 프로젝트 폴더마다 개별 가상환경을 만들지 않습니다. 루트 디렉토리에 있는 `.venv` 가상환경을 모든 프로젝트에서 공통으로 사용해 주세요.
3. **파일 참조**: 다른 프로젝트 폴더의 파일을 직접 참조하는 대신, 각 폴더 내에서 완결된 구조를 갖도록 구성합니다.
